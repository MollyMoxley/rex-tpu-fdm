03 Modèle — Mécanique de rupture supports TPU
NOUVEAU v3
En TPU, le support optimal est un problème de mécanique de rupture — pas de qualité de surface.
Objectif : minimiser la cohésion structurelle, pas maximiser la précision.
Le bon modèle : support fragile localisé → rupture propre au démontage → 0 déformation pièce élastique.
❌ MODÈLE PLASupport cohésif
Parois + interface + gap faible → structure rigide et continue.
Avec TPU : le support s'étire au lieu de casser (effet chewing-gum). Déformation pièce systématique.
✅ MODÈLE TPUSupport fragilisé
0 parois + 0 interface + gap Z/XY augmenté → structure discontinue.
Comportement fragile localisé : la rupture se concentre sur la section minimale. Détachement net.
